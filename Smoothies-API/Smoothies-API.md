*** API 5 ***

## 5. SMOOTHIES

1. **Title** : inventoryHandlerSmoothies
I.	Description : API retrieves smoothie menu items from database
II.	URL : /inventorySmoothies
III.	Method : GET
IV.	Response Codes : 200(OK), 401(Webpage doesn’t exist), 500(Internal server error)

2. **Title** : searchInventorySmoothies
I.	Description : API retrieves menu item searched by user
II.	URL : /searchInventorySmoothies
III.	Method : GET
IV.	Response Codes : 200(OK), 401(Webpage doesn’t exist), 500(Internal server error)


3. **Title** : starbucksAddToCartHandlerSmoothies
I.	Description : API adds selected items to cart
II.	URL : /addToCartSmoothies
III.	Method : PUT
IV.	Response Codes : 200(OK), 401(Webpage doesn’t exist), 500(Internal server error)


4. **Title** : cartHandlerSmoothies
I.	Description : API retrieves items  that have been added to cart
II.	URL : /cartItemsSmoothies
III.	Method : GET
IV.	Response Codes : 200(OK), 401(Webpage doesn’t exist), 500(Internal server error)


5. **Title** : starbucksProcessOrdersHandlerSmoothies
I.	Description : API processes all orders in cart
II.	URL : /processOrdersSmoothies
III.	Method : PUT
IV.	Response Codes : 200(OK), 401(Webpage doesn’t exist), 500(Internal server error)


6. **Title** : likeHandlerSmoothies
I.	Description : API processes all orders in cart
II.	URL : /likeIncreaseSmoothies
III.	Method : PUT
IV.	Response Codes : 200(OK), 401(Webpage doesn’t exist), 500(Internal server error)
